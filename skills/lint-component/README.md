# lint-component

Validates that a toolkit component follows all required conventions and schema rules.

**Invocation:** `/tw-lint-component path=skills/my-skill`

Checks: frontmatter schema, required sections, placeholder text, README presence.
Run before opening a PR with a new or modified component.

See `skill.md` for full documentation and `tests/core/lint-component/test.md` for test cases.
