# install-toolkit

AI-guided installation of tw-ai-toolkit into a consumer repository.

**Invocation:** `/tw-install-toolkit`

Walks through adding the submodule, creating `.ai/` structure, updating `CLAUDE.md`,
and optionally configuring Dependabot. For direct shell installation use `scripts/install.sh`.

See `skill.md` for full documentation and `tests/core/install-toolkit/test.md` for test cases.
